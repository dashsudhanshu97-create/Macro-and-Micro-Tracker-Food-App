# Macro-and-Micro-Tracker-Food-App

A modern, dark-themed web app to track your daily meals and nutrition — macros, all 13 vitamins, 12 minerals, omega-3, and water content.

## Features

- 🔥 **Daily Macro Rings** — Calories, Protein, Carbs, Fat, Fiber with animated progress rings
- 💊 **Complete Micronutrient Tracking** — All 13 vitamins (A, B1–B12, C, D, E, K) with RDI progress bars
- ⚡ **Mineral Tracking** — 12 minerals (Calcium, Iron, Magnesium, Phosphorus, Potassium, Sodium, Zinc, Copper, Manganese, Selenium, Chromium, Iodine)
- 💧 **Essentials** — Omega-3 and water content from food
- 🍽️ **Meal Management** — Add multiple meals, search and select foods, input quantities
- 💾 **Auto Save** — Data persists in localStorage
- 🎨 **Premium UI** — Glassmorphism dark theme with ambient blob animations

## Tech Stack

- HTML, CSS, JavaScript (Vanilla)
- CSV-based food database
- No dependencies

## Usage

1. Open `index.html` in a browser (or serve with any local server)
2. Add meals and select foods from the dropdown
3. Enter quantities to see real-time macro and micronutrient breakdowns
